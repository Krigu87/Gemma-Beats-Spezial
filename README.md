## 📲 Installation (Als App nutzen)
Für das beste Erlebnis (Vollbildmodus und dauerhafte Speicherung deiner Highscores) installiere das Spiel bitte als Web-App:

### **Für Android (Chrome)**
1. Öffne den Link in **Google Chrome**.
2. Tippe oben rechts auf die **drei Punkte** (Menü).
3. Wähle **"App installieren"** oder **"Zum Startbildschirm hinzufügen"**.

### **Für iPhone & iPad (Safari)**
1. Öffne den Link in **Safari**.
2. Tippe unten auf das **Teilen-Symbol** (Viereck mit Pfeil nach oben).
3. Wähle **"Zum Home-Bildschirm"**.
4. Tippe oben rechts auf **"Hinzufügen"**.

---

## 🎸 Features
* **Eigene Musik:** Lade deine MP3s hoch und das Spiel generiert die Beats.
* **PWA-Support:** Funktioniert wie eine echte App auf deinem Homescreen.
* **Offline-Ready:** Einmal geladen, startet die App blitzschnell.
* **Local Storage:** Deine Rekorde bleiben auf deinem Gerät gespeichert.
